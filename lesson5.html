Block-Level Elements

Block-level elements occupy the full width available, stacking on top of each other. Examples include:

<div>, <p>, <h1> to <h6>, <section>

Inline Elements

Inline elements do not start on a new line and only take up as much width as necessary. Examples include:

<span>, <a>, <strong>, <em>
